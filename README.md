# Student Management System

This is my personal student management system project built with Spring Boot and Thymeleaf. I use it to manage students, courses, and enrollments in a clean MVC-style application with a focus on learning, practice, and portfolio presentation.

I created this project to strengthen my backend and full-stack skills while working with Spring Boot, Spring Data JPA, Spring Security, MySQL, and server-rendered templates.

## What this project does

- Manage students and course records
- Enroll students into courses
- View enrollment details
- Handle authentication and authorization
- Keep the UI simple, responsive, and easy to use

## Tech Stack

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Spring Security
- Thymeleaf
- MySQL
- Maven

## Project Structure

```text
studentmanagement/
├─ pom.xml
├─ src/
└─ target/
```

## How to run

1. Open the `studentmanagement` folder in your IDE.
2. Configure your MySQL database connection in `src/main/resources/application.properties`.
3. Run the Spring Boot application from `StudentmanagementApplication.java`.
4. Open the application in your browser and start managing students and courses.
